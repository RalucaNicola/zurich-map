Old Zurich in Photos

Explore images of old Zurich on a 3D map of the city. The georeferenced, digitized images are part of ETH Library and they stem from the E-Pics platform. The map uses building data from Open Data Zurich.

The images are released under CC BY-SA 4.0 by ETH Library. The buildings are released under CC0 by the city of Zurich. The data was processed and published with ArcGIS Pro. The map was built with ArcGIS API for JavaScript . Check out the code on GitHub.
